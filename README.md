# INNOVATION-AND-PRODUCTIVITY-AN-EMPIRICAL-ANALYSIS-OF-RUSSIAN-FIRMS
This study analyzes the causal R&amp;D-TFP relationship in Russian industry (2011-2023) using 691,644 observations. The methodology progresses from reduced-form to the dynamic endogenous innovation model of Doraszelski and Jaumandreu(2013), identifying productivity drivers across eight production-oriented OKVED sections.
